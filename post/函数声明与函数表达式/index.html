函数声明与函数表达式

· 2019-06-11 ·

一道JavaScript面试题的思考

问题

偶然在网上看到这么一道面试题，现记录下来以便参考。

问：以下代码的输出结果是什么？

var a = 1;
(function a(){
a = 2;
console.log(a)})()

如果你试着在控制台执行那么得到的结果会是函数a本身

ƒ a(){
a = 2;
console.log(a)}

探讨分析

如果你知道函数声明和函数表达式的区别就很好理解了，让我们重新审视一下这道题目

var a = 1;
(function a(){
a = 2;
console.log(a)})()

首先函数a是一个立即调用函数表达式（IIFE） ，既它是一个函数表达式，而非函数声明。

函数表达式和函数声明

声明函数有以下两种方式

  • 函数声明
function foo() {
  console.log('I am foo')
}

*函数表达式

var bar = function foo() {
  console.log('I am foo')
}

匿名函数、立即执行函数也是函数表达式

// 匿名函数表达式
(function() {
  console.log('I am foo')
})()

// 立即执行函数
(function foo() {
  console.log('I am foo')
})()

在函数表达式内部有一个特殊的变量，该变量和函数名相同，指向函数本身，并无法被修改
